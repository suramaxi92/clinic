# 🏥 Clinic Management System

## 📌 Description
This is a simple Django-based web application to manage patient records, including name, age, prescriptions, and visit history.

## 🚀 Features
- Add new patients
- View patient details
- Store prescriptions
- Track last visit

## 🛠️ Tech Stack
- Python
- Django
- MySQL

## 📂 Project Setup

### 1️⃣ Clone the repository
git clone https://github.com/your-username/project-name.git
cd project-name

### 2️⃣ Install dependencies
pip install -r requirements.txt

### 3️⃣ Run migrations
python manage.py migrate

### 4️⃣ Start server
python manage.py runserver
